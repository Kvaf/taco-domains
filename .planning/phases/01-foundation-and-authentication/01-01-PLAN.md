    1. Create `prisma/schema.prisma` with SQLite datasource (for dev simplicity per PROJECT.md) and ALL models:

       **Functional in Phase 1 (Auth.js required models):**
       - User: id (cuid), email (unique), emailVerified, passwordHash, name, image, createdAt, updatedAt. Relations to Account, Session.
       - Account: id, userId, type, provider, providerAccountId, refresh_token, access_token, expires_at, token_type, scope, id_token, session_state. Compound unique on [provider, providerAccountId].
       - Session: id, sessionToken (unique), userId, expires. Relation to User.
       - VerificationToken: identifier, token (unique), expires. Compound unique on [identifier, token].

       **Defined but unused until later phases:**
       - Domain: id, name (unique), userId, tld, status (enum: ACTIVE, PENDING, EXPIRED, TRANSFERRED), registeredAt, expiresAt, autoRenew (default true), locked (default false), whoisPrivacy (default true), createdAt, updatedAt. Relation to User, DNSRecord[].
       - DNSRecord: id, domainId, type (enum: A, AAAA, CNAME, MX, TXT, NS, SRV, CAA), name, value, ttl (default 3600), priority (nullable), createdAt, updatedAt. Relation to Domain.
       - MarketplaceListing: id, domainId, sellerId, type (enum: FIXED_PRICE, AUCTION), price, minBid (nullable), auctionEndsAt (nullable), status (enum: ACTIVE, SOLD, CANCELLED, EXPIRED), createdAt, updatedAt. Relations to Domain, User (seller), Bid[].
       - Bid: id, listingId, bidderId, amount, createdAt. Relations to MarketplaceListing, User.
       - Transaction: id, domainId, sellerId, buyerId, amount, type (enum: PURCHASE, AUCTION_WIN), createdAt. Relations to Domain, User (seller), User (buyer).

       Use enums for DomainStatus, DNSRecordType, ListingType, ListingStatus, TransactionType.

    2. Create `src/lib/db/prisma.ts` — PrismaClient singleton with globalThis caching pattern for Next.js hot reload:
       ```typescript
       import { PrismaClient } from "@prisma/client";
       const globalForPrisma = globalThis as unknown as { prisma: PrismaClient };
       export const prisma = globalForPrisma.prisma || new PrismaClient();
       if (process.env.NODE_ENV !== "production") globalForPrisma.prisma = prisma;
       export default prisma;
       ```

    3. Run `npx prisma generate` and `npx prisma db push` to create the SQLite database and generate types.
